# Installing the Chrome Extension

## Part 1 - Download the Code

**1. Open the GitHub repository**  
https://github.com/rubynikal/phish  

**2. Download ZIP file**  
Click the green **Code** button (top right), then click **Download ZIP**.  
It will download to your Downloads folder.

**3. Extract the files**  
- Windows: Right-click → Extract All  
- Mac: Double-click the ZIP file  

Keep this folder somewhere permanent - Chrome needs it to stay there.

---

## Part 2 — Load into Chrome

**4. Open Extensions page**  chrome://extensions

**5. Enable Developer Mode**  
Turn on the toggle in the top-right corner.

**6. Load the extension**  
Click **Load unpacked**, then:
- Select the extracted folder  
- Click Select Folder  

---

## 🎉 Done!

- The extension should now appear in your Chrome toolbar  
- If not, click the 🧩 puzzle icon and pin it
